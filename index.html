Weather Dashboard

An interactive weather application using a third-party API

A powerful weather dashboard that provides real-time weather information for any location. The app fetches data from a reliable weather API and presents it in a clean, visual format. Users can search for any city, view current conditions, and see a multi-day forecast.

Key features include a dynamic background that changes based on the weather, search auto-complete for a better user experience, and interactive charts for a graphical view of temperature trends.

  • Developed with vanilla JavaScript for a lightweight, fast experience.
  • Integrated with OpenWeatherMap API for accurate data.
  • Used Chart.js to create dynamic and interactive data visualizations.
  • Implemented local storage to save user's favorite locations.
  • Designed a fully responsive layout for all devices.
